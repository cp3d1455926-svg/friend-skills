## 🎯 这是什么？

朋友Skill 把你的朋友的说话方式、性格、口癖提炼成 AI 数字分身。

**它不代替你的朋友**，而是在朋友不在线时，给你一个温暖的陪伴。

和朋友Skill的核心不是「完美模拟」，而是捕捉「关系温度」：
- 🗣️ 朋友怎么和你说话？（语气、口癖、表情包套路）
- 🎭 你们之间有什么独家梗？（一起吐槽过的事、互相称呼的昵称）
- 💬 TA的陪伴角色是什么？（吐槽搭子？人生导师？损友？情绪树洞？）

## 🚀 快速开始

### 1. 安装

```bash
git clone https://github.com/cp3d1455926-svg/friend-skills.git
cd friend-skill
```

### 2. 创建你的第一个朋友分身

```bash
node scripts/friend-builder.js
```

按提示输入朋友的信息，等待几秒，你就拥有了一个数字朋友！

### 3. 和朋友聊天

```bash
node scripts/chat-demo.js 张三
# 把"张三"换成你朋友的名字
```

## 📋 执行流程

### Phase 0: 朋友信息录入

用户说要「造朋友」时，先问清楚基础信息：

**必填信息：**
1. 名字/昵称（你们怎么叫TA的）
2. 性格关键词（3-5个，比如：毒舌但贴心、搞笑女、温柔学霸、暴躁老哥）

**选填信息（越多越逼真）：**
3. 聊天记录（至少20条典型对话，越多越好）
4. 共同回忆 / 特殊梗
5. 口头禅
6. 平时喜欢聊什么话题
7. TA对你的称呼

### Phase 1: 人格提炼

根据数据量选择不同模式：

| 模式 | 触发条件 | 策略 |
|------|---------|------|
| **快速捏人** | 只有名字+性格关键词 | 用默认模板生成基础人格 |
| **标准模式** | 有名字+性格+聊天记录(20-100条) | 分析数据提取人格 |
| **深度模式** | 有大量聊天记录+朋友圈+回忆 | 完整人格分析和记忆图谱 |

### Phase 2: 人格组装

将提炼结果组装成 JSON 格式的朋友人格：

```json
{
  "friend": {
    "name": "张三",
    "nickname": "三儿",
    "personality": {
      "traits": ["毒舌", "仗义", "臭屁"],
      "mbti": "ENFP",
      "energy": "高能量"
    },
    "voice": {
      "greeting": "哟，咋又来找我了",
      "farewell": "行了，跪安吧",
      "laugh": "哈哈哈哈你认真的？",
      "moodAngry": "你有病吧",
      "moodSad": "唉，算了不说了"
    },
    "relationship": {
      "yourName": "小老弟",
      "role": "损友",
      "yearsKnown": 3,
      "sharedMemories": [
        {
          "event": "上次一起通宵打游戏",
```

### Phase 3: 验证

让用户和生成的「数字朋友」聊几句，确认：

1. 语气像不像？
2. 称呼对不对？
3. 痛点准不准？
4. 有没有你们之间的特殊梗没注入？

## 🔒 隐私保护

这是最重要的部分！

- ✅ **所有数据在本地处理** —— 朋友的信息不会上传到任何服务器
- ✅ **聊天记录只用于分析** —— 分析完后可以删除原文件
- ✅ **你控制一切** —— 生成的分身文件你可以随时删除
- ⚠️ **只导你和朋友的对话** —— 不要导入群聊或包含其他人隐私的记录

## 💡 设计哲学

朋友Skill的核心不是「完美模拟」——因为真正的朋友是不可替代的。

它只做三件事：
1. **记住你们之间的温暖** —— 共同回忆、专属梗、互相的称呼
2. **模仿朋友说话的方式** —— 语气、口癖、反应习惯
3. **在你需要的时候出现** —— 不是替代，而是补充
